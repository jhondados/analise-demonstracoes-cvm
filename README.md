# 💰 Análise de Demonstrações CVM

> Análise automatizada de demonstrações financeiras de empresas listadas (CVM) com LLM. Extrai indicadores, detecta red flags e gera relatórios.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/analise-demonstracoes-cvm/

[![Vertical](https://img.shields.io/badge/Vertical-Finanças-blue)]() [![IA](https://img.shields.io/badge/IA-LLM-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**430** Empresas · **52** Indicadores · **96%** Precisão · **2min vs 2sem** Tempo

## 🧠 Tecnologias de IA
LLM · RAG · Financial Analysis

## ✨ O que faz
- Extração automática de DRE/BP/DFC
- Cálculo de 52 indicadores
- Detecção de red flags contábeis
- Relatório executivo por LLM

## 🗄️ Fonte de dados real
[CVM — Dados Abertos](https://dados.cvm.gov.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com CVM — Dados Abertos
2. **Processamento com IA** — LLM · RAG · Financial Analysis
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
